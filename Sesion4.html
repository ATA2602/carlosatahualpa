Sesión 4: Código CSS3

¿Qué es CSS3? CSS (Cascading Style Sheets / Hojas de estilo en cascada) tampoco es un lenguaje de programación, sino que es una tecnología que se emplea para darle estilo a una página web construida en HTML.

Aprendimos sobre como las etiquetas y atributos son necesarios para sincronizar CSS y HTML. Se debe cumplir cierta sintaxis para que haya sincronía entre el documento HTML y CSS, de lo contrario, de obviarse o existir errores, no se considerarán las modificaciones hechas en CSS.

Y por último, vimos las reglas de diseño y configuración con CSS, reglas aplicadas a textos. Todo esto dentro de la herencia y la cascada en CSS
